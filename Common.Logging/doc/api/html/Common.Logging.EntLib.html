Common Logging 2.0 API Reference

Common.Logging.EntLib Assembly

This assembly contains the adapter to plug Microsoft Enterprise Logging Library 3.1. into the Common.Logging infrastructure.

For an example on how to configure

  • Common.Logging to render its output to Entlib, see EntLibLoggerFactoryAdapter.
  • Entlib to render its output to Common.Logging use CommonLoggingEntlibTraceListener.
